# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/b5732c1e-0f0a-4f45-9f86-39e6c5b1fc65

## Run Locally

**Prerequisites:**  Node.js


下载整个项目：确保你下载的是包含 package.json、server.ts 和 src 文件夹的完整项目。
安装环境：确保你的电脑安装了 Node.js。
启动程序：
在项目文件夹打开终端（CMD 或 Terminal）。
输入 npm install 安装必要的组件。
输入 npm run dev 启动服务器。
访问地址：在浏览器打开 http://localhost:3000 即可看到完整带数据的界面。
